# SecureAccess
Implementação e Avaliação de um Sistema de Autenticação Multifator (MFA) 

# Project Description
O projeto tem como objetivo desenvolver uma aplicação de Autenticação Multi-Fator (MFA), uma prática
extremamente relevante tanto no contexto académico da disciplina como no meio empresarial e
tecnológico. A MFA constitui atualmente uma das medidas mais eficazes para reforçar a segurança em redes
e sistemas de informação, sendo amplamente adotada para mitigar riscos e prevenir acessos não
autorizados.
### Projeto tem em conta os seguintes objetivos:
- Desenvolver um protótipo funcional de autenticação em duas etapas, combinando credenciais
tradicionais (username + password) com um fator adicional baseado em códigos temporários
(TOTP), compatível com aplicações móveis como o Google Authenticator.
- Realizar um estudo comparativo das principais abordagens de MFA (SMS, e-mail, aplicações OTP,
chaves físicas e biometria), avaliando vantagens, limitações e cenários de aplicação.
### Tools and Techniques to Use
Python, Flask, pyotp, qrcode, Google Authenticator, SQL, Postman
